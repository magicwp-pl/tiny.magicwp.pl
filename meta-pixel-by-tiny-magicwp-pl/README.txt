== Description ==

**The simplest way to add Meta Pixel to your WordPress site.**

This plugin does exactly one thing, and it does it perfectly: it gives you a simple input field to enter your Meta Pixel ID, then automatically places the tracking code in your site's `<head>` section. That's it. Nothing more, nothing less.

**Why choose this plugin?**

* **Zero bloat** - No unnecessary features, no dashboard widgets, no admin notices, no extra CSS or JavaScript files
* **Zero spam** - No promotional content, no upsells, no "upgrade to pro" messages
* **Zero cost** - Completely free, forever. No hidden fees, no premium versions
* **Lightning fast** - Minimal code means minimal impact on your site's performance
* **Privacy focused** - Only loads the pixel code, nothing else tracks your visitors
* **Future-proof** - Built with modern PHP 8.4 standards and WordPress best practices

**What it does:**

1. Adds a settings page under "tiny.magicwp.pl" menu in your WordPress admin
2. Provides a simple text input field for your Meta Pixel ID
3. Automatically injects the Meta Pixel tracking code into your site's `<head>` section
4. Only loads on the frontend (never in admin area)

**What it doesn't do:**

* No cookie consent management
* No GDPR compliance features
* No advanced tracking options
* No analytics dashboard
* No premium features locked behind paywalls

If you need a simple, clean solution to add Meta Pixel tracking to your WordPress site without any unnecessary complexity, this is the plugin for you.

== Frequently Asked Questions ==

= Do I need to modify any theme files? =

No. The plugin automatically adds the Meta Pixel code to your site's `<head>` section. No theme modifications required.

= Will this slow down my website? =

No. The plugin is extremely lightweight and only adds the Meta Pixel script to your pages. The impact on page load time is minimal and only occurs on the frontend.

= Does this plugin collect any data? =

No. This plugin does not collect, store, or transmit any data. It only adds the Meta Pixel tracking code that you configure. All data collection is handled by Meta's own tracking script.

= Is this plugin compatible with caching plugins? =

Yes. The Meta Pixel code is added via WordPress hooks, so it works with all major caching plugins.

= Can I use this on multiple sites? =

Yes. The plugin is free to use on as many sites as you want.

= Will there be premium features or paid versions? =

No. This plugin will always be completely free with no premium versions or paid features.
