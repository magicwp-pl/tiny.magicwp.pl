# Tiny WordPress Plugins by tiny.magicwp.pl

A monorepo containing lightweight, focused WordPress plugins created for the [magicwp.pl](https://magicwp.pl) project.

## About

This repository hosts a collection of minimal WordPress plugins that do one thing and do it well. Each plugin is designed with simplicity, performance, and zero bloat in mind.

**Philosophy:**
- ✅ One purpose per plugin
- ✅ Zero bloat, zero spam, zero cost
- ✅ Modern PHP 8.4+ compatible
- ✅ WordPress coding standards compliant
- ✅ Lightweight and fast

## Plugins

### [meta-pixel-by-tiny-magicwp-pl](./meta-pixel-by-tiny-magicwp-pl/)

Simple Meta Pixel integration for WordPress. Just add your Pixel ID and it automatically appears in your site's `<head>` section.

**Features:**
- Simple settings page with Pixel ID input
- Automatic code injection into site head
- Zero bloat, zero spam, zero cost
- PHP 8.4+ compatible
